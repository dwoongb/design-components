## Usage Examples

### Using Tailwind Colors in Components

```tsx
import { Button } from '@dwoongb/design-components';

// Override component colors with Tailwind utilities
<Button className="bg-purple-500 hover:bg-purple-600">
  Purple Button
</Button>

<Button className="bg-green-500 hover:bg-green-600">
  Green Button
</Button>

// Combine with other Tailwind utilities
<Button className="shadow-lg border-2 border-blue-300 hover:scale-105">
  Styled Button
</Button>
```

### Using Tailwind Colors Directly

```tsx
// Text colors
<p className="text-blue-500">Blue text</p>
<p className="text-red-600">Red text</p>

// Background colors
<div className="bg-slate-100 p-4">
  Light background
</div>

// Border colors
<div className="border-2 border-purple-500">
  Purple border
</div>
```
