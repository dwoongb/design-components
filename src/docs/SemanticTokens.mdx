# Semantic Tokens

Component-specific tokens with semantic meaning. These tokens reference foundation tokens and define how components should look.

---

## Usage in Code

### Import

```tsx
import { semanticTokens } from 'design-components/styles';
```

### Example: Button Component

```tsx
const Button = ({ variant = 'primary', size = 'md' }) => {
  const btn = semanticTokens.button;

  return (
    <button
      css={{
        // Base styles
        ...btn.base,

        // Variant styles
        backgroundColor: btn[variant].backgroundColor,
        color: btn[variant].color,

        // Size styles
        ...btn.size[size],

        // Hover state
        '&:hover': {
          backgroundColor: btn[variant].hover.backgroundColor,
        },
      }}
    >
      Click me
    </button>
  );
};
```
